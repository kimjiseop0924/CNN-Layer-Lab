EXPERIMENT 01

테스트 샘플 수를 바꾸면 정확도가 얼마나 달라질까?

모델의 정확도는 평가에 쓴 샘플 수에 따라 흔들릴 수 있다. 샘플이 10개라면 1개 차이가 곧 10%p 차이가 된다. 다른 조건은 고정한 채 테스트 샘플 수만 바꿔, 같은 모델의 점수가 얼마나 달라지는지 측정했다.

고정 학습 2048, Conv 2  ·  변수 테스트 10 / 50 / 200 / 1000
Epoch테스트 셈플 10Test 50테스트 셈플 200테스트 셈플 1000
00.0%10.0%14.0%8.5%
170.0%56.0%56.5%54.1%
270.0%78.0%73.5%72.2%
370.0%74.0%79.0%80.7%
470.0%84.0%81.5%82.5%
590.0%88.0%82.0%82.8%
📍 핵심 관찰
테스트 셈플은 epoch 1~4 동안 정확도가 70%에 4번 고정 → 측정 단위가 10%p라 그 사이 변화가 드러나지 않음
테스트 셈플 10의 마지막 90%는 10개 중 9개 = 단 1개 차이로 만든 점프. 실력 향상이 아닌 측정 단위의 착시
같은 모델 epoch 5인데 테스트 셈플 10 → 90.0% vs 테스트 셈플 1000 → 82.8% = 7.2%p 측정 왜곡 발생
테스트 셈플 1000은 54.1 → 72.2 → 80.7 → 82.5 → 82.8 매끄럽게 수렴 → 신뢰할 수 있는 학습 곡선
📌 결론
실험 간 비교는 테스트 샘플 ≥ 1,000으로 측정해야 한다. 그보다 적으면 결과가 7%p 이상 왜곡되어 모델 차이인지 측정 노이즈인지 구분이 불가능하다.
EXPERIMENT 02

학습 데이터가 많아지면 정확도는 얼마나 오를까?

데이터를 늘리면 정확도는 어떤 곡선으로 좋아지는가? EXP 01에서 확정한 신뢰할 만한 평가 기준(Test 1000)으로, 학습 데이터만 4배씩(128 → 512 → 2048) 키워가며 같은 폭의 향상이 유지되는지 확인했다.

고정 테스트 1000, Conv 2  ·  변수 학습 128 / 512 / 2048
Epoch학습 샘플 128학습 샘플 512학습 샘플 2048
014.8%16.4%9.5%
115.3%25.5%50.4%
213.9%31.9%73.3%
318.6%40.2%81.1%
424.1%51.1%83.5%
525.7%55.4%86.4%
📍 핵심 관찰
두 번의 4배 증가가 거의 동일한 폭(+30%p)으로 점수를 올림 → 정확도는 학습 샘플에 선형 비례
학습 샘플 128은 epoch를 다 써도 25.7%에서 정체 → 데이터 부족은 epoch로 메꿀 수 없음
학습 샘플 2048은 단 1 epoch에 50.4%, 5 epoch에 86.4% 도달 → 같은 시간·같은 모델이지만 출발점부터 다름
데이터 4배당 정확도 변화 (Epoch 5)
128 → 512 (×4)+29.7 %p
512 → 2048 (×4)+31.0 %p
📌 결론
데이터 4배 ≒ 정확도 +30%p. 다음 +30%p를 얻으려면 또 4배가 필요하다. 성능 향상의 가장 강력한 수단은 데이터 확보이며, 데이터가 적을 땐 epoch를 늘려도 한계가 명확하다.
